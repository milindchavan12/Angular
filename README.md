### Getting Started
- Install Angular CLI : 
```node
npm install -g @angular/cli
```
- Create New Agnaular App
```node
ng new APM
```
- navigate to APP folder
```node
cd APM
```
- Install all required packages
```node
npm install
```
- Start running APP
```node
ng serve
```
